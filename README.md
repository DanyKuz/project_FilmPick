# ПРОЭКТ FilmPick

## 1. Папка data:

### 1. Папка covers
Хранит в себе обложки для создания фильмов в формате png

### 2. Папка primer_interface
Хранит в себе пример интерфейса(к сожалению не успел реализовать)

### 3. Папка save_covers
Сюда сохраняются обложки сохраненных фильмов

### 4. Файл films.db
Хранит в себе базу данных

## 2. Папка src
- BD.py - весь код для базы данных на python
- library.py - весь код для окна со списком фильмов на python
- menu.py - весь код для главного окна(меню) на python
- new_film.py - весь код для окна с добавлением нового фильма на python
- BD.py - весь код для выбора случайного фильма на python

## 3. Файл .gitignore
Игнорирует pycache

## 4. Файл README.md
Хранит в себе описание всех файлов и папок

## 5. Файл requirements.txt
Хранит в себе версии установленных библиотек
