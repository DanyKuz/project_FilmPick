# ПРОЭКТ FilmPick

## Функциональные возможности

### 1. Добавление и настройка новых фильмов
Возможность создания нового фильма с указанием названия, категории, возрастного ограничения, обложки, отзывы и других характеристик.

### 2. Функция выбора случайного фильма
Выбор случайного фильма по заданным критериям (например, по категории).

### 3. Список всех фильмов
Отображение всех добавленных фильмов, предпочтительно в табличном виде.

### 4. Управление фильмами
Возможность удалять существующие фильмы из списка.

### 5. Отслеживание статуса просмотра
Возможность помечать статус фильма (например: "не смотрел", "смотрел 1 раз" и т.д.).

### Главное окно (MENU)
Содержит следующие кнопки:
- **"Новый фильм"** — `QPushButton`
- **"Список фильмов"** — `QPushButton`

### При нажатии на "Новый фильм"
Открывается **третье окно**, содержащее:
- Несколько полей ввода (`QLineEdit`)
- Подписи (`QLabel`)
- Кнопки (`QButton`)
- Выпадающий список (`QComboBox` или аналогичный элемент)

### При нажатии на "Список фильмов"
Открывается **второе окно**, реализующее:
- Отображение фильмов, предположительно через `QGridLayout`
- Возможность выбора фильма

## Горячие клавиши

- Ctrl + N = Новый фильм
- Ctrl + F = Список фильмов

Хранит в себе весь код окна для третьего окна

## БД

### Таблица films

| Поле               | Тип данных       | Описание                              | Пример значения             |
|--------------------|------------------|---------------------------------------|-----------------------------|
| id                 | INTEGER          | Уникальный ID (первичный ключ)        | 1                           |
| title              | TEXT NOT NULL    | Название фильма                       | "Начало"                    |
| category           | TEXT             | Жанр фильма                           | "фантастика"                |
| age_restriction    | TEXT             | Возрастное ограничение                | "16+"                       |
| status             | TEXT             | Статус просмотра                      | "не смотрел"                |
| date_added         | TEXT             | Дата добавления (в формате YYYY-MM-DD)| "2025-11-17"                |
| rating             | INTENGER         | Рейтинг (0 - 10)                      | 8                           |
| cover              | TEXT             | Название файла с обложкой             | "cover.jpg"                 |

## Работа с файлами

### CSV

- Экспорт: Создает файл содержащий все данные о фильмах(разделенные запятыми)
- Импорт: Читает CSV файл и добавляет фильм в бд

### TXT

- Экспорт: Создает красивое оформление информации о фильме
